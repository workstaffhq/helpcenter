# Aide pour les gestionnaires

Cette zone contient toute la documentation pertinente pour les employeurs, les gestionnaires et les coordinateurs qui utilisent Workstaff pour gérer et planifier leurs équipes.

Si vous êtes un travailleur utilisant l'application mobile Workstaff, veuillez consulter la zone [Pour les travailleurs](/workers/index.md).
